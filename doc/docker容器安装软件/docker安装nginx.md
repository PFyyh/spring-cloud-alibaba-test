第一步 利用docker拉取nginx镜像

```
docker pull nginx
```

 

第二步 创建需要挂载的相应的挂载目录

```
mkdir -p nginx/conf

mkdir -p nginx/html

mkdir -p nginx/logs
```

第四步 进入到nginx文件夹

```
cd nginx/
```

docker挂载nginx

```linux
docker run --name nginx -d -p 80:80 -v $PWD/conf/nginx.conf:/etc/nginx/nginx.conf -v $PWD/html:/etc/nginx/html -v $PWD/logs:/var/log/nginx nginx
```
